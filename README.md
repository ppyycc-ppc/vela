# 环间新闻

这是一款开源且纯个人打造的手环新闻小程序，它能实现在手环上简单地访问“网易新闻”（之前想做微博发现手环访问不了API）

你可以在主页找到自己感兴趣的东西，并浏览它，也可以缓存它
